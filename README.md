# Расписание летней сессии СиПД-22.03

Готовый статический сайт для публикации на GitHub Pages.

## Что загружать на GitHub

Загрузите в репозиторий файлы из этой папки:

- index.html
- letnyaya-sessiya-sipd-22-03.ics
- .nojekyll

## Проверка после публикации

1. Открывается главная страница сайта.
2. Работает переключение светлой и тёмной темы.
3. Работает кнопка «Поделиться».
4. Работает кнопка «Скачать календарь».
5. Работает поиск.
6. Работают фильтры.
7. Таймлайн переводит к нужной карточке.
8. Карточки раскрываются по нажатию.
9. На телефоне блоки не вылезают за экран.
