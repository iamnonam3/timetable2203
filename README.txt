Сайт с вопросами к зачётам и экзаменам СиПД-22

Как открыть:
1. Откройте файл index.html в браузере.

Как загрузить на сайт:
1. Загрузите в одну папку файлы index.html, icon.png и site.webmanifest.
2. Главным файлом должен быть index.html.

Что есть:
- вопросы по основным предметам
- поиск
- скачивание Word-файлов
- печать выбранного раздела
- светлая и тёмная тема
- сворачивание выбранного предмета
- цветные метки зачёта и экзамена

Важно:
- Word-файлы уже встроены в HTML, поэтому отдельная папка с документами не нужна.
- Для иконки на главном экране iPhone файлы icon.png и site.webmanifest должны лежать рядом с index.html.
